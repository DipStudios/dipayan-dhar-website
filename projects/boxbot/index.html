BoxBot (HackFrost 3.0)

Spring 2023

Skills Used: Python, Git, Project Management, Team Collaboration, UI Design, App Development

Libraries Used: Tkinter, Custom Tkinter

Link to GitHub Link to Demo Video

Overview

  • Developed a mockup of an application for an automated box-cutting system for HackFrostNL 3.0 hackathon.
  • Used Tkinter library to develop the front-end interface to algorithm application.

Background

  • Hackathon prompt: How can we use technology to reduce the carbon footprint of Industry deliveries?
  • Came up with idea to have a way to perfectly cut cardboard boxes to reduce unused space in packaging.
  • Presented project to two rounds of judging, team came third place overall.

Project Description

  • Developed front-end user interface for inputting the dimensions of goods, including an option for fragile objects.
